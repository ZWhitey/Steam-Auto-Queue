# Plugins

plugins file format:

```json
[
  {
    "name": "YOUR_PLUGINS_NAME",
    "script": "YOUR_SCRIPT_FILE_NAME",
    "enable": "true or false", // true -> enable plugin, false -> disable plugin
    "data": {
      "EVERYTHING YOUR NEED TO PASS TO PLUGIN"
    }

  },
  {
    "name": "steam-awards-2021",
    "script": "autovote.js",
    "enable": true,
    "data": {
      "votes": [
        { "voteid": "61", "appid": "892970" },
```
